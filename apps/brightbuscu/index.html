Bright Bus

The Best Way to Get Real-Time Bus Information

What Is Bright Bus?

BrightBus is a travel utility app targeted toward universities that displays real-time information assisting students in their day-to-day activities. BrightBus allows the user to save their "favorite" stops and view bus arrival times quickly and easily. BrightBus displays the actual and scheduled information for each bus in real-time, making sure students won't be late!

Features

Search
Simply enter the stop code or search for a stop by entering the name of an intersection or location.

Favorites
Add frequently used stops to your favorites to instantly access bus arrival times.
